
# Index Tab

Create index tabs to organize your tabs

## Overview

Index Tab is a Chrome extension that helps you casually organize and manage your browser tabs.
You can easily group tabs and manage them visually.

## Key Features

- Create index pages - Display a list of tabs to the right of each index tab
- Quick switching via tab bar - Switch between indexes from the tab bar at the top of the index tab
- Drag & drop reordering - Rearrange tabs and move them between indexes with D&D
- Index tab group operations - Reorder tabs under an index all together

# Index Tab

タブを整理するためのインデックスタブを作成

## 概要

Index Tabは、ブラウザのタブをゆるく整理・管理できるChrome拡張機能です。
タブを簡単にグループ化し、視覚的に管理できます。

## 主な機能

- インデックスページを作成 - インデックスタブの右にあるタブの一覧を表示
- タブバーで素早く切り替え - インデックスタブ上部のタブバーから各インデックスに切り替え
- ドラッグ&ドロップで並び替え - D&Dでタブの並び替え、インデックス間の移動
- インデックスタブグループ操作 - 配下のタブをまとめて並び替え
